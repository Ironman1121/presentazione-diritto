# ⚖️ Il Lavoro tra Costituzione e Codice Civile

> *Un'esperienza interattiva ad alta fedeltà per l'educazione legale.*

**🌐 [Visualizza l'App Live](https://Ironman1121.github.io/presentazione-diritto/)**

---

## ✨ Funzionalità Principali

* 🌌 **Sfondo WebGL Immersivo**: Un'aurora boreale dinamica generata in tempo reale utilizzando `OGL`, accompagnata da effetti particellari fluttuanti.
* 🧭 **Navigazione Spaziale 3D**: Transizioni fluide tra le slide (overview e focus) gestite tramite `Motion` (Framer Motion), con una UI ispirata alla dock di macOS.
* 📖 **Glossario Legale Fluttuante**: Termini giuridici complessi (es. *Insubordinazione*, *Vincolo Fiduciario*, *Giusta Causa*) spiegati istantaneamente tramite tooltip interattivi contestuali.
* 📊 **Diagrammi Integrati**: Visualizzazione chiara delle procedure legali (come l'iter disciplinare) renderizzati automaticamente con `Mermaid.js`.
* 🎨 **Design Premium**: Interfaccia in stile "Glassmorphism" con una palette colori elegante (Navy scuro, Oro, Bianco sporco).

---

## 📚 Argomenti Trattati

L'applicazione copre i pilastri fondamentali del diritto del lavoro italiano:

1.  **Quadro Costituzionale:** Il fondamento democratico, i diritti e doveri civici (Artt. 1, 4, 35, 36, 37, 38).
2.  **Codice Civile:** Diligenza, obbedienza e obbligo di fedeltà (Artt. 2104, 2105).
3.  **Statuto dei Lavoratori:** I limiti al potere di controllo a distanza e le garanzie del potere disciplinare (Legge 300/1970).
4.  **Lavoro Agile (Smart Working):** Diritto alla disconnessione e sicurezza dei dati (Legge 81/2017).
5.  **Sicurezza sul Lavoro:** Obblighi e tutela della salute (D.Lgs. 81/08).
6.  **Whistleblowing:** Segnalazione di illeciti a tutela dell'interesse pubblico (D.Lgs. 24/2023).
7.  **Social Media & Reputazione:** Danno all'immagine e diritto di critica nell'era digitale.
8.  **Il Vincolo Fiduciario:** La sintesi del rapporto di fiducia tra lavoratore e azienda.

## 🛠️ Stack Tecnologico

Il progetto è costruito con tecnologie web all'avanguardia:

* **Core:** React 19 + TypeScript + Vite
* **Styling:** Tailwind CSS (v4)
* **Animazioni & UI:** Motion (Framer Motion), Lucide React
* **Grafica Avanzata:** OGL (WebGL)
* **Strumenti:** Mermaid.js (Diagrammi)
